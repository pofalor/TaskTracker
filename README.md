# Система учета рабочего времени сотрудников IT-компании

## Описание проекта
Система предназначена для автоматизированного учета рабочего времени сотрудников IT-компаний. Она позволяет отслеживать время, затраченное на задачи и проекты, управлять проектами, формировать отчеты и обеспечивает гибкие настройки для различных форматов работы (офис, удалёнка, гибрид).

## Основные возможности
- **Учет рабочего времени**:
  - Автоматический и ручной ввод времени.
  - Возможность редактирования записей.
  - Добавление комментариев к выполненным задачам.
  
- **Управление проектами и задачами**:
  - Создание и редактирование проектов.
  - Назначение задач сотрудникам.
  - Отслеживание статусов задач (в работе, на проверке, завершена).
  
- **Аналитика и отчетность**:
  - Формирование отчетов по сотрудникам, проектам и задачам.
  - Сравнение запланированного и фактического времени.
  
- **Гибкость работы**:
  - Поддержка гибридного и удаленного форматов.
  - Мобильный доступ.
  
- **Безопасность**:
  - Регистрация и авторизация пользователей.
  - Разграничение прав доступа (сотрудник, менеджер, администратор).

## Установка и запуск
### Требования
- **Серверная часть**:
  - ОС: Linux (Ubuntu Server 20.04 LTS / CentOS 7+) с Docker или Windows.
  - База данных: PostgreSQL 16+.
  - Веб-сервер: Nginx/Apache.
  - Процессор: 2+ ядра (x86-64).
  - ОЗУ: 4 ГБ.
  - Дисковое пространство: 10 ГБ.

- **Клиентская часть**:
  - ОС: Windows 10 и выше.
  - Браузер: современные версии Chrome, Firefox, Edge.

### Инструкция по установке
1. **Установка серверной части**:
   - Установите PostgreSQL и настройте базу данных.
   - Разверните веб-сервер (Nginx/Apache).
   - Склонируйте репозиторий и настройте конфигурационные файлы (`appsettings.json`).

2. **Запуск клиентской части**:
   - Откройте проект в среде разработки (например, Visual Studio).
   - Запустите клиентское приложение через Angular CLI:
     ```bash
     ng serve
     ```

## Руководство пользователя
### Вход в систему
1. Откройте страницу входа.
2. Введите email и пароль.
3. Нажмите кнопку "Войти".

### Регистрация
1. На странице входа нажмите "Зарегистрироваться".
2. Заполните обязательные поля (email, пароль, имя).
3. После успешной регистрации вы будете перенаправлены в раздел рабочих пространств.

### Рабочие пространства
- **Создание рабочего пространства**:
  - Нажмите "+" в разделе "Мои рабочие пространства".
  - Выберите тип (личное или для компании) и заполните данные.
  
- **Приглашение сотрудников**:
  - Введите email или никнейм пользователя.
  - Отправьте приглашение.

### Управление проектами и задачами
- **Создание проекта**:
  - Укажите название, описание, менеджера и даты.
  
- **Создание задачи**:
  - Назначьте задачу сотруднику, укажите тип, статус и приоритет.

### Учет времени
- **Ручной ввод**:
  - Нажмите на иконку календаря у задачи.
  - Укажите затраченное время и комментарий.
  
- **Автоматический учет**:
  - Нажмите на замочек у задачи для старта/остановки отсчета времени.

## Техническая информация
### Используемые технологии
- **Серверная часть**:
  - Язык: C# (ASP.NET Core 8).
  - База данных: PostgreSQL (Entity Framework Core).
  
- **Клиентская часть**:
  - Фреймворк: Angular 19.0.2.
  - Стили: Bootstrap 5.3.3.

### Архитектура
- Клиент-серверная модель с разделением на фронтенд (Angular) и бэкенд (ASP.NET Core).
- API для взаимодействия между клиентом и сервером.
