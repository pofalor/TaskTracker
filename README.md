# TaskTracker
Система управления задачами и рабочим временем сотрудников компаний.

Организация.
В организацию управляющий может приглашать юзеров. По эмейлу или по никнейму. При этом поиск будет работать только по полному эмейлу или никнейму(не части). Также управляющий сможет удалить юзеров из организации. Юзеры будут видеть приглашения, и смогут их принимать или отклонять. Пока юзер не примет или отклонет запрос, управляющий новый послать ему не может. 
Техническая часть: 
Будет создана сущность UserWorkspaceStatusChange, записи которой будут создаваться когда юзер будет приглашать других юзеров в свою организацию.
